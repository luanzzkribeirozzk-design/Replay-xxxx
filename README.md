# Yguix

**App para transferir replays entre Free Fire e Free Fire MAX**

## Funcionalidades
- Bypass FFM → FFN: copia replays do Free Fire MAX para o Free Fire Normal
- Bypass FFN → FFM: copia replays do Free Fire Normal para o Free Fire MAX
- Terminal com log em tempo real
- Requer Shizuku

## Como usar
1. Instale o Shizuku e ative-o
2. Abra o app e conceda permissão ao Shizuku
3. Escolha a direção do bypass e toque no botão

## Build
O APK é gerado automaticamente pelo GitHub Actions a cada push.
Acesse a aba **Actions** → **Build APK** → **Artifacts** para baixar.
